# 2. Compilación y Ejecución

## 🔧 Guía Completa de Compilación y Ejecución

### Requisitos
- **Java Development Kit (JDK)** 8 o superior
- **Terminal o línea de comandos**
- Los archivos del repositorio descargados

---

## 📥 Preparación

### 1. Descargar el Repositorio

**Opción A: Clonar con Git**
```bash
git clone https://github.com/Tresillo2017/CarritoDeCompras.git
cd CarritoDeCompras
```

**Opción B: Descargar ZIP**
1. Ir a https://github.com/Tresillo2017/CarritoDeCompras
2. Click en **Code** → **Download ZIP**
3. Descomprimir en tu computadora
4. Abrir terminal en el directorio descomprimido

### 2. Verificar Java
```bash
java -version
```

Deberías ver algo como:
```
java version "11.0.x" 2021-04-20
```

Si no funciona, instala JDK desde [oracle.com](https://www.oracle.com/java/technologies/downloads/)

---

## 🔨 Compilación

### Paso 1: Abrir Terminal en el Proyecto

**Windows:**
- Abre el explorador de archivos
- Navega a la carpeta del proyecto
- Shift + Click derecho → "Abrir terminal aquí"

**Mac/Linux:**
```bash
cd /ruta/a/CarritoDeCompras
```

### Paso 2: Compilar Todos los Archivos

```bash
javac src/*.java
```

**Salida esperada:** Nada (sin errores es bueno)

### Paso 3: Verificar Compilación

```bash
ls src/*.class
```

Deberías ver:
```
src/CarritoDeCompras.class
src/Item.class
src/Main.class
```

## 🛠️ Compilación Alternativa

### Compilar Archivos Específicos
```bash
javac src/Item.java
javac src/CarritoDeCompras.java
javac src/Main.java
```

### Compilar con Warnings
```bash
javac -Xlint:all src/*.java
```

### Compilar con Documentación
```bash
javac -cp src src/*.java
```

## 🧹 Limpieza

### Eliminar Archivos Compilados

**Windows (CMD):**
```bash
del src\*.class
```

**Windows (PowerShell):**
```powershell
Remove-Item src\*.class
```

**Mac/Linux:**
```bash
rm src/*.class
```

### Limpiar Completamente
```bash
# Eliminar todo y recompilar
rm -rf src/*.class
javac src/*.java
java -cp src Main
```

## ❌ Troubleshooting

### Error: "javac: command not found"

**Problema:** Java no está instalado o no en PATH

**Solución:**
1. Descarga JDK desde [oracle.com](https://www.oracle.com/java/technologies/downloads/)
2. Instala siguiendo las instrucciones
3. Reinicia la terminal
4. Verifica con `java -version`

---

### Error: "No se encuentra el archivo o directorio"

**Problema:** La ruta está incorrecta

**Solución:**
1. Verifica que estés en el directorio correcto
2. Confirma que existe la carpeta `src/`
3. Usa ruta completa: `javac C:\ruta\completa\src\*.java`

---

### Error: "clase Main no encontrada"

**Problema:** No estás en el directorio correcto al ejecutar

**Solución:**
```bash
# Asegúrate de estar en la raíz del proyecto
java -cp src Main
```

---

### Error: "Archivos .class no se crean"

**Problema:** Problemas de permisos o errores de compilación

**Solución:**
1. Verifica que tengas permisos de escritura en `src/`
2. Revisa si hay errores de sintaxis:
   ```bash
   javac src/*.java 2>&1  # Mostrar todos los errores
   ```

## 💡 Tips Útiles

### Compilación Rápida
Crea un script bash/batch:

**Linux/Mac (run.sh):**
```bash
#!/bin/bash
javac src/*.java && java -cp src Main
```

**Windows (run.bat):**
```batch
@echo off
javac src\*.java
java -cp src Main
```

Luego:
```bash
./run.sh    # Mac/Linux
run.bat     # Windows
```

### IDE Recomendados
Si no quieres compilar desde terminal:
- **VS Code** (con Java Extensions)
- **IntelliJ IDEA** (Community Edition)
- **Eclipse** (Gratuito)
- **NetBeans** (Gratuito)
