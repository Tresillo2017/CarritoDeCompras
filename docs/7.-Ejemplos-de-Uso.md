# 7. Ejemplos de Uso

## 💻 Ejemplos Prácticos del Carrito de Compras

Una colección de ejemplos reales de cómo usar el carrito en diferentes escenarios.

## Ejemplo 1: Carrito Simple

### Código
```java
CarritoDeCompras carrito = new CarritoDeCompras();

// Agregar items
carrito.agregarItem(new Item("Manzana", 1.50, 3));
carrito.agregarItem(new Item("Pan", 2.99, 1));

// Mostrar carrito
carrito.imprimirDetalle();
```

### Salida
```
----------------------------------------
Detalles del carrito de compras:
----------------------------------------
Manzana - €1.50 x 3 = €4.50
Pan - €2.99 x 1 = €2.99

----------------------------------------
Total: €7.49 (Total de items: 4)
----------------------------------------
```

**Aprenderás:** Crear carrito, agregar items, mostrar detalle

## Ejemplo 2: Agregar Cantidad del Mismo Item

### Código
```java
CarritoDeCompras carrito = new CarritoDeCompras();

// Agregar 2 camisetas
carrito.agregarItem(new Item("Camiseta", 20.99, 2));

// Agregar 3 camisetas más
// Automáticamente se suma la cantidad

carrito.imprimirDetalle();
```

### Salida
```
Camiseta - €20.99 x 5 = €104.95

Total: €104.95 (Total de items: 5)
```

**Aprenderás:** Cómo se manejan items duplicados (suma automática)

## Ejemplo 8: Obtener Información del Carrito

### Código
```java
CarritoDeCompras carrito = new CarritoDeCompras();

carrito.agregarItem(new Item("Camiseta", 20.99, 2));
carrito.agregarItem(new Item("Pantalones", 34.99, 1));
carrito.agregarItem(new Item("Zapatos", 89.99, 1));

// Consultar información
System.out.println("Tipos diferentes: " + carrito.obtenerNumeroArticulos());
System.out.println("Total de items: " + carrito.obtenerCantidadTotal());
System.out.println("Total a pagar: €" + String.format("%.2f", carrito.calcularTotal()));

// Obtener item específico
Item item = carrito.obtenerItem("Camiseta");
System.out.println("Camiseta en carrito: " + item);
```

### Salida
```
Tipos diferentes: 3
Total de items: 4
Total a pagar: €166.97
Camiseta en carrito: Camiseta - €20.99 x 2 = €41.98
```

**Aprenderás:** Consultar información del carrito

## Ejemplo 10: Simulación de Compra Completa

### Código
```java
CarritoDeCompras carrito = new CarritoDeCompras();

System.out.println("=== TIENDA EN LÍNEA ===\n");

// Fase 1: Agregar productos
System.out.println("Fase 1: Agregando productos...\n");
carrito.agregarItem(new Item("Monitor", 249.99, 1));
carrito.agregarItem(new Item("Teclado", 149.99, 1));
carrito.agregarItem(new Item("Mouse", 79.99, 2));
carrito.imprimirDetalle();

// Fase 2: Cambiar de idea
System.out.println("\nFase 2: Cambiar cantidad de mice...\n");
carrito.actualizarCantidad("Mouse", 1);
carrito.imprimirDetalle();

// Fase 3: Agregar más
System.out.println("\nFase 3: Agregar mousepad...\n");
carrito.agregarItem(new Item("Mousepad", 29.99, 1));
carrito.imprimirDetalle();

// Fase 4: Arrepentirse
System.out.println("\nFase 4: Remover mousepad...\n");
carrito.removerItem("Mousepad");
carrito.imprimirDetalle();

// Resumen
System.out.println("\n=== RESUMEN FINAL ===");
System.out.println("Artículos: " + carrito.obtenerNumeroArticulos());
System.out.println("Total items: " + carrito.obtenerCantidadTotal());
System.out.println("Total: €" + String.format("%.2f", carrito.calcularTotal()));
```

### Salida (abreviada)
```
=== TIENDA EN LÍNEA ===

Fase 1: Agregando productos...

Monitor - €249.99 x 1 = €249.99
Teclado - €149.99 x 1 = €149.99
Mouse - €79.99 x 2 = €159.98

Total: €559.96

... (más fases)

=== RESUMEN FINAL ===
Artículos: 3
Total items: 3
Total: €479.98
```

**Aprenderás:** Flujo completo de compra

## Ejemplo 11: Patrón Seguro con Try-Catch

### Código
```java
public class GestorCarrito {
    
    public void agregarItemSeguro(CarritoDeCompras carrito, 
                                   String nombre, double precio, int cantidad) {
        try {
            Item item = new Item(nombre, precio, cantidad);
            carrito.agregarItem(item);
            System.out.println("✓ " + nombre + " agregado");
        } catch (IllegalArgumentException e) {
            System.out.println("✗ Error al agregar: " + e.getMessage());
        }
    }
    
    public void removerItemSeguro(CarritoDeCompras carrito, String nombre) {
        try {
            carrito.removerItem(nombre);
            System.out.println("✓ " + nombre + " removido");
        } catch (IllegalArgumentException e) {
            System.out.println("✗ Error al remover: " + e.getMessage());
        }
    }
}

// Uso
GestorCarrito gestor = new GestorCarrito();
CarritoDeCompras carrito = new CarritoDeCompras();

gestor.agregarItemSeguro(carrito, "Laptop", 999.99, 1);
gestor.agregarItemSeguro(carrito, "Mouse", -30.00, 1);  // Falla
gestor.removerItemSeguro(carrito, "Monitor");  // Falla
```

### Salida
```
✓ Laptop agregado
✗ Error al agregar: El precio debe estar entre 0.01€ y 1000000.00€...
✗ Error al remover: El artículo 'Monitor' no existe en el carrito
```

**Aprenderás:** Patrón de manejo seguro de excepciones

## Ejemplo 12: Carrito Multicategoría

### Código
```java
CarritoDeCompras carrito = new CarritoDeCompras();

// Electrónica
carrito.agregarItem(new Item("Smartphone", 699.99, 1));
carrito.agregarItem(new Item("Headphones", 149.99, 2));

// Ropa
carrito.agregarItem(new Item("Camiseta", 29.99, 3));
carrito.agregarItem(new Item("Jeans", 79.99, 1));

// Libros
carrito.agregarItem(new Item("Java Programming", 45.99, 2));

// Mostrar resumen
System.out.println("=== RESUMEN DE COMPRA ===\n");
carrito.imprimirDetalle();

System.out.println("\n=== ESTADÍSTICAS ===");
System.out.println("Categorías diferentes: " + carrito.obtenerNumeroArticulos());
System.out.println("Total de unidades: " + carrito.obtenerCantidadTotal());
System.out.println("Monto total: €" + String.format("%.2f", carrito.calcularTotal()));
```

### Salida
```
=== RESUMEN DE COMPRA ===

Smartphone - €699.99 x 1 = €699.99
Headphones - €149.99 x 2 = €299.98
Camiseta - €29.99 x 3 = €89.97
Jeans - €79.99 x 1 = €79.99
Java Programming - €45.99 x 2 = €91.98

Total: €1261.91 (Total de items: 9)

=== ESTADÍSTICAS ===
Categorías diferentes: 5
Total de unidades: 9
Monto total: €1261.91
```

**Aprenderás:** Manejo de múltiples categorías

## 🎯 Patrones de Uso Recomendados

### Patrón 1: Crear y Verificar
```java
Item item = new Item("Producto", 25.00);
if (item != null) {
    carrito.agregarItem(item);
}
```

### Patrón 2: Validar Antes de Modificar
```java
Item item = carrito.obtenerItem(nombreItem);
if (item != null) {
    carrito.actualizarCantidad(nombreItem, nuevaCantidad);
}
```

### Patrón 3: Try-Catch para Operaciones Críticas
```java
try {
    carrito.removerItem(nombreItem);
} catch (IllegalArgumentException e) {
    System.err.println("Error: " + e.getMessage());
}
```
