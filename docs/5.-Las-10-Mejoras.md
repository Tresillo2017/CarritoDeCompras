# 5. Las 10 Mejoras

## 📊 Resumen Visual de las 10 Mejoras

Una refactorización sistemática transformó un carrito básico en código profesional.

## 1️⃣ Separación de Programa Principal

### ❌ Antes
```
CarritoDeCompras
├── main() ← El programa está aquí
├── agregarItem()
├── removerItem()
└── imprimirDetalle()
```

**Problema:** La clase tiene dos responsabilidades

### ✅ Después
```
Main ← Programa independiente
├── main()
└── Demostraciones

CarritoDeCompras ← Solo lógica
├── agregarItem()
├── removerItem()
└── imprimirDetalle()
```

**Ventaja:** Separación clara, reutilizable

**Impacto:** 🟢 Alto - Arquitectura más limpia

## 2️⃣ Separación de Responsabilidades

### ❌ Antes
```java
public void imprimirDetalle() {
    // Itera items
    for (Item item : items) {
        System.out.println(item.nombre + " - " + item.precio);
    }
    
    // Calcula total
    double total = 0;
    for (Item item : items) {  // ← DUPLICACIÓN
        total += item.precio;
    }
    
    // Imprime total
    System.out.println("Total: " + total);
}
```

### ✅ Después
```java
public double calcularTotal() {
    // Una sola responsabilidad
    double total = 0;
    for (Item item : items.values()) {
        total += item.calcularPrecioTotal();
    }
    return total;
}

private void imprimirListaArticulos() {
    // Una sola responsabilidad
    for (Item item : items.values()) {
        System.out.println(item.toString());
    }
}

public void imprimirDetalle() {
    // Orquesta las responsabilidades
    imprimirListaArticulos();
    imprimirTotal();
}
```

**Ventaja:** Métodos pequeños, testables, reutilizables

**Impacto:** 🟢 Alto - Código más limpio

## 4️⃣ Encapsulación y Validación

### ❌ Antes
```java
class Item {
    public String nombre;      // ❌ Público
    public double precio;      // ❌ Público
    
    public Item(String nombre, double precio) {
        this.nombre = nombre;  // Sin validación
        this.precio = precio;  // Sin validación
    }
}

// Permitía esto:
Item item = new Item(null, -100.00);  // ¡Inválido pero permitido!
```

### ✅ Después
```java
class Item {
    private String nombre;     // ✅ Privado
    private double precio;     // ✅ Privado
    
    public void setNombre(String nombre) {
        if (nombre == null || nombre.trim().isEmpty()) {
            throw new IllegalArgumentException("Nombre vacío");
        }
        this.nombre = nombre;
    }
    
    public void setPrecio(double precio) {
        if (precio < 0.01 || precio > 1_000_000.00) {
            throw new IllegalArgumentException("Precio fuera de rango");
        }
        this.precio = precio;
    }
}

// Ahora esto falla:
Item item = new Item(null, -100.00);  // ✅ Excepción clara
```

**Ventaja:** Imposible crear Items inválidos

**Impacto:** 🟢 Alto - Datos siempre válidos

## 5️⃣ Eliminación de Duplicación

### ❌ Antes
```java
// Iteramos dos veces (líneas 23-27 y 29-33)
for (Item item : items) {
    System.out.println(item.nombre + " - $" + item.precio);
}

double total = 0;
for (Item item : items) {  // ← Iteración duplicada
    total += item.precio;
}
```

### ✅ Después
```java
// Método reutilizable
public double calcularTotal() {
    double total = 0;
    for (Item item : items.values()) {
        total += item.calcularPrecioTotal();
    }
    return total;
}

// Usado en múltiples lugares sin duplicación
double total = calcularTotal();
System.out.println("Total: " + total);
```

**Ventaja:** Una sola fuente de verdad

**Impacto:** 🟡 Medio - Mantenimiento más fácil

## 6️⃣ Manejo Robusto de Excepciones

### ❌ Antes
```java
public void removerItem(Item item) {
    items.remove(item);  // ¿Se removió o no? 🤷
}

// Uso
carrito.removerItem(itemInexistente);  // Falla silenciosamente
```

### ✅ Después
```java
public void removerItem(String nombreItem) {
    if (nombreItem == null || nombreItem.trim().isEmpty()) {
        throw new IllegalArgumentException("Nombre inválido");
    }
    
    if (!items.containsKey(nombreItem)) {
        throw new IllegalArgumentException(
            "El artículo '" + nombreItem + "' no existe");
    }
    
    items.remove(nombreItem);
}

// Uso
try {
    carrito.removerItem("Inexistente");
} catch (IllegalArgumentException e) {
    System.out.println("Error: " + e.getMessage());
}
```

**Ventaja:** Errores detectados y controlados

**Impacto:** 🟢 Alto - Robustez del sistema

### ❌ Antes

```java
private static final String MONEDA = "€";

```

**Ventaja:** Cambios centralizados, código más claro

**Impacto:** 🟡 Medio - Mantenibilidad

## 8️⃣ Método toString() Personalizado

### ❌ Antes
```java
// Lógica de formateo dispersa en CarritoDeCompras
System.out.println(item.nombre + " - $" + item.precio + " x" + cantidad);
```

### ✅ Después
```java
// En Item.java
@Override
public String toString() {
    return String.format("%s - €%.2f x %d = €%.2f",
        nombre, precio, cantidad, calcularPrecioTotal());
}

// En CarritoDeCompras.java - Uso simple
System.out.println(item.toString());
```

**Salida:**
```
Camiseta - €20.99 x 2 = €41.98
```

**Ventaja:** Item controla su formato

**Impacto:** 🟡 Medio - Código más limpio

## 9️⃣ Escalabilidad Mejorada (HashMap)

### ❌ Antes - ArrayList
```java
List<Item> items = new ArrayList<>();  // O(n)

// Búsqueda: O(n)
for (Item item : items) {
    if (item.nombre.equals(nombreBuscado)) {
        return item;
    }
}

// Remoción: O(n)
items.remove(item);  // Busca primero, luego remueve
```

### ✅ Después - HashMap
```java
Map<String, Item> items = new HashMap<>();  // O(1)

// Búsqueda: O(1)
Item item = items.get(nombreBuscado);  // Instantáneo

// Remoción: O(1)
items.remove(nombreBuscado);  // Instantáneo
```

### Comparativa de Rendimiento

| Operación | ArrayList | HashMap | Mejora |
|-----------|-----------|---------|--------|
| Buscar 1 item | O(n) | O(1) | ∞x |
| Con 1,000,000 items | ~1s | ~0.001s | 1000x |

**Ventaja:** Escalable a millones de items

**Impacto:** 🟢 Alto - Rendimiento crítico

## 🔟 Soporte para Cantidades

### ❌ Antes
```java
// Para agregar 5 camisetas, hay que crear 5 Items
carrito.agregarItem(new Item("Camiseta", 20.99));
carrito.agregarItem(new Item("Camiseta", 20.99));
carrito.agregarItem(new Item("Camiseta", 20.99));
carrito.agregarItem(new Item("Camiseta", 20.99));
carrito.agregarItem(new Item("Camiseta", 20.99));
// ❌ Duplica items en la lista
```

### ✅ Después
```java
// Constructor con cantidad
Item item = new Item("Camiseta", 20.99, 5);

// Agregar más del mismo tipo suma automáticamente
carrito.agregarItem(new Item("Camiseta", 20.99, 3));
// Total: 8 camisetas en UN item

// Actualizar cantidad
carrito.actualizarCantidad("Camiseta", 10);

// Cálculo automático
item.calcularPrecioTotal();  // 20.99 × 10 = 209.90
```

**Ventaja:** Intuitivo y eficiente

**Impacto:** 🟢 Alto - Funcionalidad mejorada
